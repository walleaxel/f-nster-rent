# Fönster Rent

En professionell webbplats för fönsterputs tjänster.

## Om Fönster Rent

Vi erbjuder professionell fönsterputs för hem, lägenheter och företag med:
- Snabb service
- Fasta priser
- Nöjda kunder

## Paket

### Bas - 299 kr
- Utvändig fönsterputs
- Upp till 10 fönster
- Perfekt för mindre bostäder

### Standard - 599 kr
- Invändig & utvändig puts
- Upp till 20 fönster
- Mest populära valet

### Premium - 999 kr
- Komplett rengöring
- Karmar och detaljer
- Upp till 40 fönster

## Filer

- `index.html` - Huvudsida med HTML-struktur
- `style.css` - Stilar och layout
- `script.js` - JavaScript för formulärhantering

## Hur man använder

1. Öppna `index.html` i en webbläsare
2. Bläddra genom paket och recensioner
3. Fyll i bokningsformuläret för att boka en tid
